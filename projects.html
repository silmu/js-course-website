1. Z-index

  When elements are positioned, they can overlap other elements.
The z-index property specifies the stack order of an element (which element should be placed in front of, or behind, the others).
An element can have a positive or negative stack order.
Example:

p {
position: absolute;
left: 0px;
top: 0px;
z-index: 10;
}

This project shows different overlapping div elements. Z-index is increased by 10 for each element.

3. Date

This project shows the current date and time with the following code:

<script>
var d = new Date();
document.write(`${d.getDate()}/${d.getMonth()}/${d.getFullYear()}`);
document.write(`${d.getHours()}:${d.getMinutes()}:${d.getSeconds()}`);
</script>

The time is updated every second with the following code added in tag:

<meta http-equiv="refresh" content="1">
